# Lowell Contracting Website

A responsive, single-page static website for Lowell Contracting.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.

GitHub will display the public website URL after deployment finishes.

## Contact form

The estimate form opens the visitor's default email app and prepares an email to `brycelowell21@gmail.com`. This avoids requiring a server or paid form service.

## Edit company information

Contact details and website text are in `index.html`. Colors and layout are in `styles.css`.
